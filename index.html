Featured Project

This project is the centerpiece of my current portfolio and reflects my interests in causal inference, health policy, and applied statistical analysis.

Did Medicaid Expansion Increase Preventive Care?

Causal evidence from BRFSS using modern machine learning methods.

This figure highlights the relationship between Medicaid expansion and preventive care outcomes. The project uses staggered difference-in-differences, event study methods, and Double Machine Learning to examine whether expansion improved access to preventive care.
